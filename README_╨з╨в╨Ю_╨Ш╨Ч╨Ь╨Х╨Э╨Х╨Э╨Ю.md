# Что изменено в проекте

## Верхнее меню
- `Портфолио` переименовано в `Работы`.
- `Текстиль` переименован в `Изделия`.
- `Услуги` переименованы в `Печать`.
- Выпадающее меню `Печать` разложено на группы:
  - Технологии печати
  - Печать на изделиях
  - Дополнительно

## Новые страницы
Добавлены и наполнены контентом SEO-страницы:
- `/termopechat/` — Термопечать
- `/shelkografiya/` — Шелкография
- `/sublimaciya/` — Сублимация
- `/pechat-na-polo/` — Печать на поло

## Изделия
В раздел `Изделия` добавлено `Поло`.
Добавлены карточки поло в каталог изделий.

## Шаблон страниц
`ServiceContentPage.jsx` расширен:
- добавлен блок примеров применения;
- добавлен блок важных нюансов;
- добавлены кастомные таблицы цен;
- оставлены текущие цены DTF из проекта;
- для термопечати добавлен ориентир по ценам термопереноса.

## Главная
Карточки услуг на главной теперь отражают реальные технологии:
- DTF печать
- Шелкография
- Термопечать
- Сублимация
- Подготовка макета

## Важное
Цены на шелкографию и сублимацию оставлены как индивидуальный расчёт, потому что точного прайса в проекте не было.
