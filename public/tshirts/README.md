# Фото футболок

Здесь лежат реальные фотографии футболок для карточек товаров.

## Структура папок

- `oversize/180/` — оверсайз футболки 180 г/м²
- `oversize/240/` — оверсайз футболки 240 г/м²
- `classic/180/` — базовые футболки 180 г/м²

Внутри каждой папки плотности используются подпапки по цветам.

Примеры:

- `oversize/180/pink/`
- `oversize/180/dark-gray/`
- `oversize/180/melange/`
- `oversize/240/black/`
- `oversize/240/white/`
- `oversize/240/pink/`
- `oversize/240/beige/`
- `classic/180/black/`
- `classic/180/white/`

## Как загружать фото

Фото не привязаны к типу кадра. Галерея берёт изображения просто по порядку файлов:

- `1.jpg` или `01.jpg`
- `2.jpg` или `02.jpg`
- `3.jpg` или `03.jpg`

И так далее, максимум до 12 фото в одной папке.

## Поддерживаемые форматы

- `jpg`
- `jpeg`
- `png`
- `webp`
- `avif`

## Как менять порядок

Просто переименуйте файлы. Например, то, что называется `1.jpg`, будет показано первым.

## Если фото нет

Если в папке нет реальных изображений, сайт автоматически покажет графическую заглушку.
